 Online Examination System (Java)

A simple **console-based Online Examination System** developed using **Core Java**. This project allows users to log in, update their profile, change their password, take a multiple-choice online exam, view their score, and log out.

## 🚀 Features

- 🔐 User Login Authentication
- 👤 View and Update Profile
- 🔑 Change Password
- 📝 Multiple Choice Online Exam
- 📊 Automatic Score Calculation
- 🚪 Logout System
- 💻 Console-Based Interface

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Scanner Class
- Arrays
- Methods
- Classes & Objects

## ▶️ How to Run

1. Clone the repository.

```bash
git clone https://github.com/your-username/OnlineExamSystem.git
```

2. Open the project in your Java IDE (IntelliJ IDEA, Eclipse, NetBeans, or VS Code).

3. Compile the project:

```bash
javac *.java
```

4. Run the application:

```bash
java Main
```
